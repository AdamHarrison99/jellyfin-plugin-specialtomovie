General

Logs all hard links that would be created without actually creating them. Disable to activate real linking.

Deletion Behavior

When the TV episode is removed, its linked movie folder is also deleted.
Deleting either item removes both. Without this, only removing the TV episode removes the linked movie.
Warning: This is an experimental feature. When a linked movie is deleted, the original episode file will be permanently removed along with any file or folder in the same directory whose name starts with the episode's filename. This includes subtitles, images, NFO, and trickplay data, but will also remove any other file that happens to share the same name prefix — even if it is not a Jellyfin companion file. This cannot be undone.
